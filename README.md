# 👤 Программа для регистрации пользователей

Простая программа для регистрации новых пользователей. Она запрашивает логин и пароль и сохраняет их в словарь с проверкой на уникальность логина.

## 🚀 Запуск программы

Для запуска программы убедитесь, что на вашем компьютере установлен Python версии 3.6 или выше.

1. Скачайте файлы проекта на ваш компьютер.
2. Откройте терминал или командную строку.
3. Перейдите в директорию с файлами программы.
4. Запустите программу командой:

```bash
python registration.py
```

## 💻 Использование программы

После запуска программы вам будет предложено:
- Ввести логин нового пользователя
- Ввести пароль нового пользователя

Программа автоматически проверит, существует ли пользователь с таким логином, и добавит нового пользователя, если логин уникален.

## 🛠️ Функционал программы

- ✅ Отображение списка существующих пользователей
- ✅ Проверка уникальности логина
- ✅ Добавление нового пользователя
- ✅ Защита от перезаписи существующих пользователей
- ✅ Информативные сообщения о результате операции
